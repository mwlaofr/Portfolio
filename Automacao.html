Automação

No curso de Automação, aprendi a utilizar diversas ferramentas e tecnologias que são essenciais para o desenvolvimento de sistemas automatizados. Meu aprendizado abrangeu desde conceitos básicos até a simulação de projetos com Arduino. Aqui estão alguns dos tópicos e habilidades que adquiri.

Comandos Básicos

  • pinMode(pin, OUTPUT): Configura o pino especificado como saída
  • digitalWrite(pin, valor): Define o estado do pino (HIGH ou LOW)
  • digitalRead(pin): Lê o valor do pino digital (HIGH ou LOW)
  • delay(ms): Pausa a execução do programa por um período de tempo especificado em milissegundos
  • Serial.begin(baudrate): Inicia a comunicação serial com a taxa de transmissão especificada
  • Serial.println(message): Envia dados pela porta serial seguido por uma nova linha

Arduino

O Arduino é uma plataforma de prototipagem eletrônica de código aberto baseada em hardware e software fáceis de usar. É amplamente utilizado em projetos de automação devido à sua simplicidade e versatilidade. Com o Arduino, é possível controlar luzes, motores, sensores e outros componentes eletrônicos de maneira programável. Nos meus projetos, utilizei o Arduino para simular um sistema de semáforo, onde controlei a sequência de luzes (vermelho, amarelo e verde) de forma programada. Também desenvolvi um projeto para controlar a intensidade de um LED, um display numérico para mostrar contagens e números, além de utilizar sensores de luz para criar sistemas que respondem a diferentes níveis de luminosidade. O Arduino permite uma vasta gama de aplicações, desde projetos simples até sistemas mais complexos.
Site utilizado para programação e visualização: Tinkercad - [https://www.tinkercad.com](https://www.tinkercad.com/)

Simulação de Semáforo

Desenvolvemos um sistema de controle de semáforo utilizando LEDs para representar as luzes verde, amarela e vermelha. O projeto envolveu a programação do Arduino para alternar entre as luzes em intervalos regulares, simulando o funcionamento real de um semáforo em um cruzamento.

Componentes Utilizados:

  • Placa Arduino UNO
  • Protoboard
  • Leds
  • Resistores de 220 Ohms

Introdução

Na disciplina de Linguagem de Programação, focamos no aprendizado da linguagem Java, abordando diversos conceitos e estruturas fundamentais para a programação. Aprendemos sobre variáveis, operadores, estruturas de decisão e repetição, matrizes, vetores, funções e introdução a Programação Orientada a Objeto.

Variáveis

Variável é o nome utilizado para definir um ou mais valores que são manipulados pelos programas durante a sua operação. O termo “variável” é utilizado por ser um tipo de conteúdo que pode apresentar diferentes valores enquanto o sistema está em execução.


                int idade = 17;
                double altura = 1.65;
                char genero = 'F';
                String nome = "Millena";

Operadores

Os operadores são símbolos que instruem o compilador ou o interpretador a realizar operações matemáticas, lógicas ou de manipulação de dados. Eles são fundamentais na programação, pois permitem a execução de cálculos, a comparação de valores e a tomada de decisões.

Aritméticos

Os operadores aritméticos são utilizados para realizar operações matemáticas em programas.


                int soma = 10 + 5;
                int subtracao = 10 - 5;
                int multiplicacao = 10 * 5;
                int divisao = 10 / 5;
                int resto = 10 % 3;

Lógicos

Os operadores lógicos são usados para executar operações lógicas em programas, permitindo a combinação e a manipulação de expressões booleanas para tomadas de decisão complexas.


                (n1 && n2) // 'E' lógico
                (n1 || n2) // 'OU' lógico
                (n1 ! n2) // inverte

Comparação

Os operadores de comparação são utilizados para comparar valores em programas.


              (n1 < n2) // menor
              (n1 > n2) // maior
              (n1 <= n2) // menor igual
              (n1 >= n2)  // maior igual
              (n1 == n2)  // igual
              (n1 != n2)  // diferente

Atribuição

Os operadores de atribuição são utilizados para atribuir valores a variáveis.


      int resultado = 5;    // resultado é 5
      resultado += 3;       // resultado é 8
      resultado -= 3;       // resultado é 5
      resultado *= 3;       // resultado é 15
      resultado /= 3;       // resultado é 5
      resultado %= 3;       // resultado é 2

Estrutura de Decisão

1. If e Else

Utilizada para tomar decisões com base em condições. O bloco de código dentro do “if” é executado somente se a condição for verdadeira, caso contrário, o bloco “else” (caso haja um) é executado.


    public class Main {
        public static void main(String[] args) {
            double n1;
            Scanner in = new Scanner(System.in);
            System.out.println("Digite um numero real:");
            n1 = in.nextDouble();
            if (n1 >= 0){ //se n1 for maior ou igual a 0, roda o bloco de código.
                double raiz = Math.sqrt(n1);
    
                System.out.println("A raiz quadrada de " + n1 + " é: " + raiz);
            }
            else { //se não, executa esse bloco de código.
                System.out.println("Valor inválido");
            }
        }
    }

2. Switch

Utilizada para selecionar um bloco de código a ser executado dentre várias opções, com base no valor de uma expressão. É útil quando há várias condições que dependem do valor de uma única variável.


   public class Main {
       public static void main(String[] args) {
           int dia;
           Scanner in = new Scanner(System.in);
           System.out.println("Digite um número de 1 a 7");
           dia = in.nextInt();
   
           String diaSemana= "";
           switch (dia) {
               case 1:
                   diaSemana = "domingo";
                   break;
               case 2:
                   diaSemana = "segunda-feira";
                   break;
               case 3:
                   diaSemana = "terça-feira";
                   break;
               case 4:
                   diaSemana = "quarta-feira";
                   break;
               case 5:
                   diaSemana = "quinta-feira";
                   break;
               case 6:
                   diaSemana = "sexta-feira";
                   break;
               case 7:
                   diaSemana = "sabado";
                   break;
               default:
                   diaSemana = "Número inválido";
                   break;
           }
   
           System.out.println("O dia da semana correspondente ao número " + dia + " é " + diaSemana);
       }
   }

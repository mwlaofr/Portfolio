Linguagem de Programação

Na disciplina de Linguagem de Programação, focamos no aprendizado da linguagem Java, abordando diversos conceitos e estruturas fundamentais para a programação. Aprendemos sobre variáveis, operadores, estruturas de decisão e repetição, matrizes, vetores, funções e introdução a Programação Orientada a Objeto.

Variáveis

Variável é o nome utilizado para definir um ou mais valores que são manipulados pelos programas durante a sua operação. O termo “variável” é utilizado por ser um tipo de conteúdo que pode apresentar diferentes valores enquanto o sistema está em execução.


                int idade = 17;
                double altura = 1.65;
                char genero = 'F';
                String nome = "Millena";

Operadores

Os operadores são símbolos que instruem o compilador ou o interpretador a realizar operações matemáticas, lógicas ou de manipulação de dados. Eles são fundamentais na programação, pois permitem a execução de cálculos, a comparação de valores e a tomada de decisões.

Aritméticos

Os operadores aritméticos são utilizados para realizar operações matemáticas em programas.


            int soma = 10 + 5;
            int subtracao = 10 - 5;
            int multiplicacao = 10 * 5;
            int divisao = 10 / 5;
            int resto = 10 % 3;

Lógicos

Os operadores lógicos são usados para executar operações lógicas em programas, permitindo a combinação e a manipulação de expressões booleanas para tomadas de decisão complexas.


              (n1 && n2) // 'E' lógico
              (n1 || n2) // 'OU' lógico
              (n1 ! n2) // inverte

Comparação

Os operadores de comparação são utilizados para comparar valores em programas.


              (n1 < n2) // menor
              (n1 > n2) // maior
              (n1 <= n2) // menor igual
              (n1 >= n2)  // maior igual
              (n1 == n2)  // igual
              (n1 != n2)  // diferente

Atribuição

Os operadores de atribuição são utilizados para atribuir valores a variáveis.


      int resultado = 5;    // resultado é 5
      resultado += 3;       // resultado é 8
      resultado -= 3;       // resultado é 5
      resultado *= 3;       // resultado é 15
      resultado /= 3;       // resultado é 5
      resultado %= 3;       // resultado é 2

Estrutura de Decisão

1. If e Else

Utilizada para tomar decisões com base em condições. O bloco de código dentro do “if” é executado somente se a condição for verdadeira, caso contrário, o bloco “else” (caso haja um) é executado.


    public class Main {
        public static void main(String[] args) {
            double n1;
            Scanner in = new Scanner(System.in);
            System.out.println("Digite um numero real:");
            n1 = in.nextDouble();
            if (n1 >= 0){ //se n1 for maior ou igual a 0, roda o bloco de código.
                double raiz = Math.sqrt(n1);
    
                System.out.println("A raiz quadrada de " + n1 + " é: " + raiz);
            }
            else { //se não, executa esse bloco de código.
                System.out.println("Valor inválido");
            }
        }
    }

Explicação: O usuário entrará com um número e, caso o mesmo seja um número real, o programa imprime sua raiz quadrada, entrando na condição IF. Caso o número digitado não seja real, ele segue para o else e imprime para o usuário que o valor digitado é inválido.

2. Switch

Utilizada para selecionar um bloco de código a ser executado dentre várias opções, com base no valor de uma expressão. É útil quando há várias condições que dependem do valor de uma única variável.


   public class Main {
       public static void main(String[] args) {
           int dia;
           Scanner in = new Scanner(System.in);
           System.out.println("Digite um número de 1 a 7");
           dia = in.nextInt();
   
           String diaSemana= "";
           switch (dia) {
               case 1:
                   diaSemana = "domingo";
                   break;
               case 2:
                   diaSemana = "segunda-feira";
                   break;
               case 3:
                   diaSemana = "terça-feira";
                   break;
               case 4:
                   diaSemana = "quarta-feira";
                   break;
               case 5:
                   diaSemana = "quinta-feira";
                   break;
               case 6:
                   diaSemana = "sexta-feira";
                   break;
               case 7:
                   diaSemana = "sabado";
                   break;
               default:
                   diaSemana = "Número inválido";
                   break;
           }
   
           System.out.println("O dia da semana correspondente ao número " + dia + " é " + diaSemana);
       }
   }

Explicação: O usuário digita um número de 1 a 7 e, através do switch, comparando o valor da variável “dia” com cada case, o programa imprime o dia da semana correspondente ao número digitado. O comando “break” é usado para sair do switch após executar o bloco de código correspondente. O “default” é executado caso nenhum dos cases corresponder ao valor digitado.

Estrutura de Repetição

For

Utilizado para repetir um bloco de código um número específico de vezes.


    for (int i = 0; i < 10; i++) {
        System.out.println("Valor de i: " + i);
    }

Explicação: O loop “for” inicializa a variável i em 0, depois verifica se seu valor é menor que 10, e incrementa i a cada repetição, imprimindo o valor de i a cada passo. O loop encerra quando o valor de i passar a ser maior que 10.

While

A estrutura de repetição while permite a criação de um loop que executa um bloco de instruções repetidas vezes enquanto uma condição for verdadeira


    int i = 0;
    while (i < 10) {
        System.out.println("Valor de i: " + i);
        i++;
    }

Explicação: O loop while continua executando o bloco de código enquanto a condição “i < 10” for verdadeira, incrementando “i” a cada repetição.

Estrutura de Dados

Vetores

Tipos que permitem o armazenamento de uma coleção de valores do mesmo tipo, ou seja, os vetores são estruturas de dados homogêneas (formados por elementos de mesmo tipo) e unidimensionais.


    // Declarando um vetor de inteiros com 5 elementos
    int[] numeros = new int[5];
    
    // Inicializando o vetor com valores
    numeros[0] = 10;
    numeros[1] = 20;
    numeros[2] = 30;
    numeros[3] = 40;
    numeros[4] = 50;

    // Acessando e exibindo o terceiro elemento do vetor
System.out.println("O terceiro elemento é: " + numeros[2]);

Matrizes

Tipo de dado usado para representar uma certa quantidade de variáveis de valores homogêneos. Uma matriz é uma série de variáveis do mesmo tipo referenciadas por um único nome, onde cada variável é referenciada através de um índice entre colchetes. Armazena elementos em grade bidimensional ou superior.


        int[][] matriz = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };
        System.out.println(matriz[0][1]);
        // Imprime o elemento na primeira linha,
        // segunda coluna: 2

Funções

Bloco de código que executa alguma operação. Opcionalmente, uma função pode definir parâmetros de entrada que permitem que os chamadores passem argumentos para a função. Uma função também pode retornar um valor como saída.


            public static int soma(int a, int b) {
                return a + b;
            }
            
            public static void main(String[] args) {
                int resultado = soma(5, 3);
                System.out.println("Soma: " + resultado); // Imprime: Soma: 8
            }

Nesse exemplo, a função “soma” é uma forma de encapsular a lógica de somar dois números em um único bloco de código reutilizável. Primeiro recebe dois parâmetros do tipo inteiro (a e b) e, no corpo da função (return a + b), realiza a soma dos dois parâmetros e retorna o resultado.

Programação Orientada a Objeto

A Programação Orientada a Objetos (POO) é um paradigma que organiza o software em torno de "objetos" em vez de funções e lógica. Um objeto é uma instância de uma classe, que é uma estrutura que define as propriedades (atributos) e comportamentos (métodos) que o objeto pode ter.

Conceitos Trabalhados

1. Classes e Objetos

Uma classe é uma forma de definir um tipo de dado em uma linguagem orientada a objeto. Ela é formada por dados e comportamentos. Para definir os dados são utilizados os atributos, e para definir o comportamento são utilizados métodos. Depois que uma classe é definida podem ser criados diferentes objetos que utilizam a classe.

Objeto é uma instância de uma classe. Cada objeto pode ter diferentes valores para os atributos definidos na classe.


    public class Pessoa {
        // Atributos da classe
        private String nome;
        private int idade;
    
        // Construtor da classe
        public Pessoa(String nome, int idade) {
            this.nome = nome;
            this.idade = idade;
        }
    
        // Método da classe
        public void apresentar() {
            System.out.println("Olá, meu nome é " + nome + " e eu tenho " + idade + " anos.");
        }
    }
    
    public class Main {
        public static void main(String[] args) {
            // Criação de um objeto da classe Pessoa
            Pessoa pessoa = new Pessoa("João", 25);
            // Chamada do método apresentar do objeto pessoa
            pessoa.apresentar(); // Imprime: Olá, meu nome é João e eu tenho 25 anos.
        }
    }

2. Encapsulamento

É o princípio de esconder os detalhes internos de um objeto e permitir o acesso apenas através de métodos públicos. Isso protege os dados e garante a integridade dos atributos.


        public class Pessoa {
            private String nome;
            private int idade;
        
            // Métodos getter e setter para o atributo nome
            public String getNome() {
                return nome;
            }
        
            public void setNome(String nome) {
                this.nome = nome;
            }
        
            // Métodos getter e setter para o atributo idade
            public int getIdade() {
                return idade;
            }
        
            public void setIdade(int idade) {
                this.idade = idade;
            }
        }

Atributos Privados: nome e idade são privados, acessíveis apenas dentro da classe Pessoa.
Métodos Públicos: getNome( ), setNome( ), getIdade( ), setIdade( ) são usados para acessar e modificar os atributos privados.

3. Herança

É um princípio de orientação a objetos, que permite que classes compartilhem atributos e métodos, através de "heranças". Ela é usada na intenção de reaproveitar código ou comportamento generalizado ou especializar operações ou atributos.


        // Superclasse
        class Animal {
            void emitirSom() {
                System.out.println("O animal faz um som");
            }
        }
        
        // Subclasse
        class Cachorro extends Animal {
            // A subclasse herda o método emitirSom da superclasse
            void correr() {
                System.out.println("O cachorro está correndo");
            }
        }
        
        public class Main {
            public static void main(String[] args) {
                // Criando um objeto da subclasse
                Cachorro meuCachorro = new Cachorro();
        
                // Chamando o método herdado da superclasse
                meuCachorro.emitirSom(); // Saída: O animal faz um som
        
                // Chamando o método da subclasse
                meuCachorro.correr();    // Saída: O cachorro está correndo
            }
        }

Superclasse: Animal tem um método emitirSom( ) que imprime uma mensagem genérica.
Subclasse: Cachorro herda de Animal e tem seu próprio método correr( ).
Reutilização de Código: No método main, criamos um objeto da classe Cachorro. Este objeto pode usar tanto o método herdado emitirSom( ) quanto o método específico correr( ) da subclasse.
